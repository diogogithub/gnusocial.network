Una red social, federada y libre

Únete! Instalar

GNU social es un software para la web que puedes usar para tener tu propia red social, tanto pública como privadamente.

  • El software soporta tanto instalaciones comunitarias como de una sola cuenta, y se puede usar en una red local o formando parte de la red social federada GNU social.
  • El software se viene usando desde hace cinco años y es muy estable. GNU social también se comunica fácilmente con otros servidores GNU social, y con redes sociales tradicionales como Twitter.
  • Al estar escrito con los estándares web de facto PHP y MySQL, GNU social se puede ejecutar prácticamente en cualquier sitio que albergue otras aplicaciones web, como WordPress o Drupal.
  • Y, ya que GNU social forma parte del proyecto GNU, es 100% software libre, sin características maliciosas, spyware ni publicidad.

Quem desenvolve o GNU social?

A equipa atual de desenvolvimento é liderada por Diogo Cordeiro. Uma listagem contendo todos os programadores atuais e outros colaboradores encontra-se disponível aqui. Os fundadores do projeto do GNU social foram Matt Lee, Evan Prodromou e Mikael Nordfeldth.
